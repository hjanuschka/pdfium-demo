PDFium JXL-in-PDF Demo (Rust jxl-rs)

Demo artifacts for a bring-up implementation that adds JPEG XL image decoding to PDFium using the Rust decoder jxl-rs. This repo contains sample PDFs that embed JXL image XObjects using the (draft/leader) filter name /JXLDecode, plus the rendered PNG outputs produced by pdfium_test. Animated JXL is rendered as frame 0 only.

/Filter /JXLDecode · Rust-only decoder · alpha via synthesized /SMask · animation → frame 0

Dice (alpha)

RGBA JXL embedded in PDF. PDFium decodes alpha and synthesizes an /SMask. Background is magenta to visualize transparency.

Download PDF Rendered PNG

anim-icos (animated JXL → frame 0)

Animated RGBA JXL embedded in PDF. PDFium renders only the first frame (magenta background included).

Download PDF Rendered PNG

Build note: this demo corresponds to a local PDFium build with pdf_enable_rust_jxl=true and a Rust toolchain available. The filter name /JXLDecode is used as a bring-up placeholder while the spec is evolving.
